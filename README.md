# Menu PWA

Статичний односторінковий сайт для вибору страв. Дані при першому запуску записуються в `localStorage`, далі сайт працює з локальним сховищем браузера.

## Локальний запуск

```bash
python3 -m http.server 8000 --bind 0.0.0.0
```

Після цього відкрити:

```text
http://localhost:8000/
```

## Можливості

- вибір страв по тегах;
- збереження вибору в `localStorage`;
- окремий режим `Редагувати меню`;
- додавання, редагування й видалення страв;
- редагування інгредієнтів у формі страви;
- редагування тегів;
- імпорт та експорт даних JSON-файлом;
- архів видалених страв з можливістю відновити або очистити архів;
- службові `id` для страв та інгредієнтів.

## Дані

Поточний ключ сховища:

```text
menu-pwa.database.v8
```

Окремого довідника тегів немає: список тегів будується динамічно з поля `tags` кожної страви.

Режим редагування не запамʼятовується: після перезавантаження відкривається головний екран.

## Структура

```text
index.html
assets/css/styles.css
assets/js/storage.js
assets/js/i18n.js
assets/js/app.js
i18n/uk.json
```

GitHub Pages може хостити сайт з кореня репозиторію через `index.html`.
